# Descripción 
Es una aplicación web multiplataforma la cual fue diseñada con el proposito de ver series, peliculas y programas de televesión, tambien esta diseñada de manera intuitiva y atractiva. Esta pagina web esta diseñada con HTML, CSS y JS donde tiene funciona con una API pública de TVMaze para ofrecer información sobre miles de títulos disponibles.

Los usuarios pueden navegar por el catálogo completo de series, buscar títulos especificos donde se pueden filtar por género y consultar su detalle completo de cada programa, además cuenta con un sistema de favoritos persistente que permite guardar los titulos de preferencia y acceder a ellos en cualquier momento, junto al historial de busquedad para remotar consultas anteriores.
